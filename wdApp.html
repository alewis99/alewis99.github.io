Web Design Program

Python Code used to build websites

From a basic school project to an app that can develop a real website

Back in college, during the python segment of my class, I decided to look ahead, and look into dealing with files in python. Since we were a little behind, the class never made it to python files, so I had to figure that out on my own. It wasn't too complex to grasp the concept, since going about it was very similar to how I dealt with files in C++ (without having to import any libraries, which made it easier, since we can just use files). Of course, you had to open the files, then close the files in order to not ruin the program. This does not mean that this was a simple project.

In the version I made in the classroom, it was simple, since it was just basic. Pick some colors for the basic CSS, select if user wants a picture, write a header and an article, write those into an HTML file, and there's the website. But now, since i'm making something to create a real website, I had to really think about how I was going to make the program make a realistic website. I first needed a template to go off of, so that it is easier to keep things in order.

The Functions

getTitle

This function is to prompt the ser to enter a title. On an earlier verion of this, this also served as the logo.

openCSS

This function is to open the CSS file and write the first couple of lines, so that we can go straight into the design aspect.

makeTheme

The overall color patterns for the body text and background is decided here.

makeTool

This function creates the basic HTML for the nav bar. It prompts the users to enter how many links they want, and after each iteration, the link name is sent to an array.

writeTop

This funtion is where the nav bar on top is designed.

getEntry

This is the function that creates the content for the page.

closeHTML

I wrote this, so that the last lines of the HTML are in before the files are closed. The CSS codes has already been concluded in a previous function.
